# bexruzdev — Bekhruz Tursunboev Portfolio

Personal portfolio website for Bekhruz Tursunboev, specializing in business automation, custom Telegram bots, CRM dashboards, and API integrations in Uzbekistan.

## Tech Stack & Design System
* **HTML5 & Vanilla CSS**: Clean, semantic markup. Strict CSS layout tokens.
* **Pure JavaScript**: Bilingual dictionary-based translation switcher, lightweight screenshot carousels, responsive touch-swipe support, and keyboard arrow navigation.
* **Design Theme**: Premium Utilitarian Minimalism & Editorial UI. Features serif typography contrast, warm paper tone surfaces, concentric bezel ratios, and macOS-style custom mockup frames.

## Key Features
* **Bilingual Switcher**: Native Uzbek (default) and Russian translations.
* **Screenshot Carousels**: Replaced resource-heavy terminal animations with clean, high-resolution website screenshots.
* **Pricing Calculator**: Interactive starting cost calculator for business services (Landing pages, Telegram bots, CRM dashboards, payment systems).
* **Responsive Layout**: Fluid layouts, single-column stacks on tablets, and touch-optimized controls for mobile viewports.
